Support

Technische Deep-Dive

🖥️ Overzicht

  • Machine: Support (HackTheBox)
  • Moeilijkheidsgraad: Easy
  • Platform: Windows (Active Directory domein)
  • IP-adres gebruikt in write‑up: 10.129.43.78
  • Belangrijke diensten: SMB (139/445), LDAP (389), WinRM (5985)

🔍 Recon & Enumeration

  • nmap -sC -sV -oN support.nmap 10.129.43.78
  • Open poorten: 139/445 (SMB), 389 (LDAP), 5985 (WinRM)
  • SMB anonieme lijst: share support-tools gevonden
  • Download van UserInfo.exe.zip via smbclient
  • Uitpakken en voorbereiden voor decompilatie

🚪 Initial Access

  • ILSpy gebruikt om UserInfo.exe te decompileren
  • XOR‑versleuteld LDAP wachtwoord ontdekt in class Protected
  • Python script decrypt.php (XOR met key 'armando' en 0xDF) geeft LDAP password
  • ldapsearch met gebruiker ldap@support.htb en gevonden wachtwoord
  • Wachtwoord van gebruiker support gevonden in AD info‑veld → evil‑winrm login als support

🔐 Privilege Escalation

  • whoami /all toont SeMachineAccountPrivilege en lidmaatschap van Shared Support Accounts
  • PowerView.ps1 toont GenericAll rechten van deze groep op de Domain Controller
  • Powermad.ps1 aangemaakt machineaccount FakeComputer
  • RBCD geconfigureerd via msds-allowedtoactonbehalfofotheridentity met SID van FakeComputer
  • Rubeus.exe S4U ticket → impacket secretsdump → Administrator NTLM hash → Pass‑the‑Hash evil‑winrm

🛠️ Tools & Technieken

  • nmap – poort‑ en service‑detectie
  • smbclient – anonieme SMB shares benaderen
  • ILSpy – .NET decompilatie op Linux
  • Python – XOR decrypt script
  • ldapsearch – LDAP enumeratie
  • evil-winrm – Windows shell via WinRM
  • PowerView.ps1 & Powermad.ps1 – AD enumeration en machine‑account aanmaken
  • Rubeus.exe – Kerberos ticket manipulatie (S4U)
  • impacket – ticketConverter en secretsdump
  • Pass-the-Hash – authenticatie met NTLM hash

💡 Geleerde lessen

  • .NET code is niet echt verborgen – ILSpy decompileert volledig naar leesbare C#
  • AD info‑velden zijn leesbaar voor alle authenticated gebruikers – geen wachtwoorden daar opslaan
  • GenericAll over een computer‑object (vooral DC) geeft volledige domein controle
  • SeMachineAccountPrivilege lijkt onschuldig maar, met GenericAll, leidt tot domain takeover via RBCD
  • Wachtwoorden met $ tekens vereisen enkele quotes in bash om variabele‑expansie te voorkomen
  • Evil‑WinRM kan Kerberos tickets van Rubeus niet gebruiken voor SMB – gebruik impacket op eigen machine
  • Aangemaakte machine accounts kunnen verdwijnen bij HTB‑reset – maak ze opnieuw bij elke sessie
