🕵️ What is ethical hacking?

  • Imagine hiring a locksmith to test whether your locks actually work — that's ethical hacking
  • Security experts search for weaknesses in computer systems with permission
  • The goal is to find problems before real criminals do
  • The machine 'Sau' is a practice vault where hackers can test their skills

🖥️ The system: what did we find?

  • We found a server with two 'doors': one publicly accessible, one locked from the inside
  • The open door was a kind of digital mailbox that forwards messages — called 'request-baskets'
  • Behind the locked door ran an email security program called 'Mailtrail'
  • Both programs were outdated — like using an old lock whose master key is already online

🎯 The weak spots: three domino pieces

  • 🚪 Domino 1: The mailbox could be abused to open the locked inner door — like asking a receptionist to deliver a private letter on your behalf
  • 💉 Domino 2: The mail program accepted poisoned input — imagine writing a command in a login form and the computer obediently executes it
  • 👑 Domino 3: An admin command accidentally opened a text reader with all-powerful privileges — as if a notepad turned out to be the key to the vault
  • Together these three flaws created a chain reaction from outsider to full control

💡 What did we learn?

  • One weak point is already dangerous, but a chain of small mistakes enables a complete takeover
  • Updating software is like replacing your locks when a master key is in circulation — do it on time
  • Access rights should be as limited as possible: give no one more keys than strictly necessary
  • Don't rely on a firewall alone — if someone is already inside, there must be a second line of defence

🏢 Why does this affect you and your organisation?

  • Many companies run forgotten, outdated software — just like this system — without knowing it
  • One seemingly innocent program can be the back door through which attackers enter
  • Regular testing by ethical hackers prevents criminals from being the first to find the weak spot
  • Cybersecurity is not just an IT problem — it protects customer data, reputation, and business continuity
